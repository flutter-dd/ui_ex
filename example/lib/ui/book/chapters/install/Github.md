# GitHub加速
## 下载[SwitchHosts](https://swh.app/zh/)
## 安装好`SwitchHosts`后
## 1. 新建一个本地的规则把自己系统的Hosts复制过来做备份
### - 方案名：local（可以自行命名）
## 2. 新建一个远程的规则：
### - 方案名：GitHub（可以自行命名）
### - 类型：远程
### - URL 地址：https://cdn.jsdelivr.net/gh/ineo6/hosts/hosts
### - 自动更新：1 小时（可以自行调整）
## 这样就可以和仓库中最新的hosts保持同步。
